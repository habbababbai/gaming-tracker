# Mobile

Mobile app for tracking game backlog. Expo (React Native), file-based routing.

## Commands

| Command | Description |
|---------|-------------|
| `bunx expo start` | Dev server + QR |
| `bunx expo run:ios` | iOS simulator |
| `bunx expo run:android` | Android emulator |

## Structure

- `app/` – routes (file-based)
- `components/` – reusable UI
- `constants/` – theme, config
